# Information Transmission in Crime Branch Using Steganography

## Project Overview
This project is a secure image-based data transmission system developed using steganography techniques. It enables confidential information to be hidden within digital images and securely transmitted.

## Features
- Secure data hiding within images
- Two-level authentication
- Encrypted communication
- Data confidentiality and integrity
- Image quality preservation

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Apache (WAMP Server)

## How to Run

1. Start WAMP Server.
2. Copy the project folder to the `www` directory.
3. Import the database into phpMyAdmin.
4. Start Apache and MySQL services.
5. Access the application through:
    http://localhost/Steganography/stegano/
